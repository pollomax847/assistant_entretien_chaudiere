# Assistant Entretien Chaudière

Interface web responsive pour techniciens ENGIE ou indépendants.  
Permet de calculer automatiquement :

- Puissance chauffage (formule G × V × (19 – Text))
- Volume et pression vase d’expansion
- Puissance réelle via compteur gaz
- Débit eau chaude sanitaire
- Checklist d’entretien chaudière

## Déploiement

1. Cloner ou téléverser les fichiers (`index.html`, `style.css`, `script.js`) dans un dépôt GitHub.
2. Déployer gratuitement sur [Vercel](https://vercel.com) :
   - Lancer un nouveau projet
   - Sélectionner ce dépôt
   - C’est tout !

## Utilisation

- Fonctionne en ligne ou en local (PC et mobile)
- Aucun compte requis
- Zéro base de données
